Checklist Manual de Testes
Painel de Monitoramento - UI da Tabela de Dados

Objetivo

Validar se a tabela da seção "Prévia dos Dados" está com nomes de colunas descritivos e design moderno, conforme o padrão PEERS.

Checklist

Instruções de Teste

  • Abra o painel em diferentes dispositivos e navegadores.
  • Verifique cada item do checklist acima, marcando conforme for validado.
  • Teste diferentes filtros e métricas para garantir que os nomes das colunas mudam corretamente.
  • Durante o carregamento dos dados, observe se o feedback visual está presente.
  • Observe se o design segue o padrão visual do layout principal da PEERS.

Resultado

Se todos os itens estiverem marcados, a implementação está conforme esperado para UI da tabela de dados.
