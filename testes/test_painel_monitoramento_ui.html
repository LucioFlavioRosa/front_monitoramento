Checklist Manual de Teste

  1. Os nomes das colunas estão visíveis e corretos: Projeto, Usuário, Modelo, Dia, Valor.
  2. O design da tabela segue o padrão moderno e fluido do layout do Assessment (bordas arredondadas, cores, fontes, sombra, hover).
  3. A tabela é responsiva: em telas pequenas, cada célula mostra o nome da coluna via data-label e layout "card".
  4. O hover nas linhas do corpo da tabela aplica gradiente suave.
  5. A formatação dos dados nas células está correta (valores numéricos, datas, nomes).
  6. Testar em diferentes tamanhos de tela (desktop, tablet, mobile).
  7. Testar navegação por teclado (tab, foco, acessibilidade).

Para testes automatizados, recomenda-se usar Selenium, Playwright ou Cypress para validar os critérios acima.
